How this works

These are SVG placeholder plants with the correct component structure for animation. Each plant has named groups: pot-body, face (with eye-left, eye-right, mouth, blush), and foliage (with individual leaf-N elements). Leaves sway independently with staggered CSS animations. Faces swap expressions by toggling visibility on pre-drawn mouth/eye variants. Click a plant to see the bounce interaction. Use the buttons to change all expressions at once.

When the illustrator delivers final SVGs with the same group naming convention, they drop in as replacements with zero animation code changes.
